## 📋 About

Institutional website created to showcase services, completed projects and make it easy for clients to get in touch via WhatsApp. The design prioritizes elegance and sophistication, aligned with the high-quality craftsmanship offered.

## ✨ Features

- Fully responsive layout (mobile, tablet and desktop)
- Animated mobile drawer menu
- Scroll reveal effect on sections
- Instructions
- Design UI UX
- Smooth pure CSS animations

## 🛠️ Tech Stack

- Semantic HTML5
- CSS3 (variables, grid, flexbox, animations)
- Vanilla JavaScript
- Google Fonts (Cormorant Garamond + DM Sans)
